# Image Detection Pipeline: From Simple NN to VGG-16

This project demonstrates an image detection pipeline, moving from custom data collection to basic Neural Networks, custom CNNs, and advanced transfer learning with VGG-16.

## 📌 Project Overview
* **Objective:** [e.g., Detecting and classifying object types]
* **Dataset:** Custom dataset collected and cropped manually.

---

## 🛠️ Project Stages

1. **Data Collection & Preprocessing**: Images were collected, cropped to focus on the target subjects, and resized for model compatibility.
2. **Simple Neural Network**: A basic fully connected baseline network to establish initial performance.
3. **Custom CNN Detector**: A Convolutional Neural Network designed to automatically extract spatial image features.
4. **VGG-16 Transfer Learning**: Utilizing a pre-trained VGG-16 network architecture to maximize detection accuracy.
